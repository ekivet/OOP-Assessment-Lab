1. Describe the difference between a Class and an Object.

A class is a binding of methods and variables in a single scope.  An Object is an instance of a class.


2. What keyword do you use to turn a normal Class into a Subclass? (choose one)
  a. implements
  b. extends
  c. uses
  d. sub

(b)


3. Describe the difference between public and private?

public access modifiers allow other programmers to manipulate your code.  with private modifiers access is limited to other members of the same class. 


4. Can a subclass access a private method of its parent class? (choose one)
  a. Yes
  b. No

(b)


5. Define the term constructor.

constructors lay the blueprint to creating objects



6. Can you access a static method of a class without making an instance of it? (choose one)
  a. Yes
  b. No

(a)


7. Describe the purpose of overriding a method.

overriding a method is useful when you have a child class invoking the same method as a parent class but you want it to follow a different set of instructions.
